Warp — Fast, Private File Beaming

Warp is a pocket-size P2P portal that moves your files directly between phones, tablets, and laptops — no cloud hop, no size cap, and no waiting for a shared Wi-Fi network. Files travel end-to-end encrypted via QUIC with TLS 1.3. Nothing hits a server. Zero accounts. Zero analytics.

View on GitHub ⬇ Download coming soon

🪐 Speed-dial contacts

Pin up to eight favourites as 3-D planets. Drag any file onto their world and it's on its way.

🔒 End-to-end encrypted

Transfers travel over modern QUIC with TLS 1.3. A tiny relay kicks in only if your router blocks direct connections.

💤 Works while the other phone sleeps

Warp queues the send, wakes the receiver with a silent push, and resumes from the last byte if either side drops out.

📱 iOS · Android · macOS · Windows

A sleek sci-fi UI on top of a high-performance Rust core (Iroh). One download, every platform.

👤 Zero accounts

Each device is its own identity — an Ed25519 public key. No sign-up, no tracking, no analytics.

♾️ No size cap

4K video clips, folder archives, project assets — Warp handles them all without upload limits.

Typical moments Warp fixes

  • Shoot a 4K clip on your phone and drop it straight onto your desktop editor before the coffee cools.
  • Hand a wedding-photo folder to your parents without teaching them cloud-drive permissions.
  • Sync project assets with a colleague on hotel Wi-Fi without hoping the upload finishes overnight.
